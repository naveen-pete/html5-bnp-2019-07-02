Boldening text - font-weight

Font weight represents the degree of boldness/lightness of a font. The value 'normal' (for font-weight) represents normal font boldness. Similarly 'bold' makes the text bold. Values 100, 200, 300, ..., 800, 900 represents varying degrees of boldness. Value 400 is equivalent to norml and 700 is equivalent to bold. However only custom fonts support font variants of various degrees of boldness.
