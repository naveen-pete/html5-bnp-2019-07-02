Setting line height - line-height

The line-height property can be used with absolute (eg. px) units, relative units (eg. %, em etc.), or in unitless fashion. When used with %, it is a % of the computed font-size. When used unitless, it represents the factor to multiply the computed font-size by (similar to %). However, due to differences in the way the line-height property value is inherited, it is better to use unitless value for line height.
Reference: Eric Meyer's article on value with relative unit vs unitless value for line-height
