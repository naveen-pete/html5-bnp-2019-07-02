Setting font size - font-size

Font sizes can be specified in absolute (px, cm etc.) or relative (%, em, rem stc.) units. For example, when specified in em unit, the font size inherited from parent is multiplied by the given number. For rem, the given number is multiplied by the base font-size (font-size of either html element - this is called root element - 'r' in rem stands for root).
