Setting fonts - font-family

A group of fonts (called "font stack") can be specified for an element. The first available font is picked from the stack and used. Fonts are grouped into "families" like serif, sans-serif, monospace, cursive etc. Usually a generic font-family is specified as the final fallback font (when none of the fonts from the font stack is available, one belonging to the font family is automatically picked). Font names with multiple words need to be quoted (example, "Helvetica Neue"). Usually only fonts from same family will be specified in a font stack.
