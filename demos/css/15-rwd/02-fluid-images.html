RWD - Fluid Layout

Fluid layout is a Responsive Web Design (RWD) technique where horizontal dimensions (widths, horizontal paddings, borders & margins) are specified in percentages (%). Since borders can't be specified in percentages, box-sizing: border-box is normally used when borders are present for elements. As a result of specifying the widths etc. in percentages, the box for all elements (and thus the resulting layout) scales with the size of the screen.
