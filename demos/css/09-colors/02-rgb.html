The RGB Model

Colors can be represented as a mixture of red, green and blue color values. In CSS, each of these can take a value between 0 - 255 (2^8). Thus the combinations can produce 2^24 unique colors.

Values specified in hexadecimal number system - #rgb (one hex digit [0-f] per color)
