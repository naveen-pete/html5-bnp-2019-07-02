The background-repeat property

If the background image width or height is less than the element's width and height (padding box along with right and bottom borders), the background image will by default be tiled in one or both directions (vertical, horizontal) in order to fill up the entire box (border box). This behavior can be controlled using the background-repeat property, so that tiling occurs only in one direction instead, or in neither direction. The first tile's top-left coordinates corresponds to the padding box top-left coordinates.
Aside: Unlike "border box" there is no term called "padding box" as padding-box isn't a valid value for the box-sizing property.
