Inheritance of styles

Most CSS properties get inherited by children from the parent element. However some of them are not.
Note: It is qute intuitive which are and which ones are not. The 'color' property for example is inherited as you'd expect child element text content to have same color as parent text content. Border and padding on the other hand would have an unpleasant effect if inherited and you don't want it inherited - so it isn't by default.
