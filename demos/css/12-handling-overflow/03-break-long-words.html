Handling horizontal overflow - break words in between

Lines are usually broken at word boundaries. A long word can thus overflow its container horizontally. This behavior of breaking lines at word boundaries only can be overcome using the word-wrap property set to the value break-word - this causes a word to potentially break in between and continue on the following line.
