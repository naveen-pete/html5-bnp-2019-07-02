Box Model - The Border Area

The border area is the space for border. The border style, color and width for each direction can be adjusted individually. Border widths can be specified only in absolute units (eg. px). There are intermediate shorthand properties like border-top, border-right etc. (to set up border width, color, style for a direction), border-style, border-color, border-width (to set up a border-related feature for all directions), and the overall "border" shorthand property (to set up a uniform border for all directions). Box shadows can be specified using the box-shadow property (CSS3).
