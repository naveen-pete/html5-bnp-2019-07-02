Introduction to the Box Model - The Element Box

Every element gets a rectangular space (box) in the rendered document where the content of the element, its borders etc. go in. The boxes are allocated in source order starting from the top of the document. The width of each box depends on whether its display is inline, block etc. and explicit settings for the different dimensions of the box.

A box has 4 parts. Some CSS properties related to each part are mentioned.

  • Content area - width, height, max-width, min-height etc.
  • Padding - padding-top, padding (shorthand) etc.
  • Border - border-top-style, border-width (shorthand), border (shorthand) etc.
  • Margin - margin-right, margin (shorthand) etc.
