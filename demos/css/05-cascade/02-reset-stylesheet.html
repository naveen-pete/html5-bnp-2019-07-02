Reset Stylesheet

The browser's default styling is usually reset by the author in order to start off with a uniform set of styles across browsers. There are many reset stylesheets available for free. One of them popular ones is the Eric Meyer reset stylesheet. We use a reset stylesheet inspired by the Eric Meyer reset stylesheet - https://github.com/murtaugh/HTML5-Reset/blob/master/assets/css/reset.css


Observe that this page has no margins for body, boldening for headings, margins for paragraphs, bullets for lists etc.
