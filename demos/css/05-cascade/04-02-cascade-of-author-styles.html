Cascade order of author styles

Multiple author stylesheets (external stylesheets) can be included in a document. Also, styles may be embedded in a document in multiple style tags (embedded styles). All of these cascade in source order (in the order they appear). However inline styles get more preference and trump all other author styles.
Note: We have used the same selector throughout (p). This has an effect on which style wins when there us conflict. We shall cover rules regarding this in the section on selector specificity.

Embedded and external stylesheets get applied in source order. Conflicting property values get cascaded.
