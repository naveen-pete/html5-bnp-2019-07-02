Selector Specificity - Introduction

As long as the selector used is the same, the order decides the final styles. However the selector plays a role in conflict resolution of property value. A more "specific" selector like id gets more importance than class selector. Similarly a class is considered to be more specific than the element selector, and hence gets more importance.
